# 📊 FocusForge

### Personal Learning Analytics & Mastery System

> Track. Analyze. Optimize. Master.

---

## 🧠 Overview

FocusForge is a full-stack learning intelligence system that transforms your daily study routine into actionable insights across Quant, IELTS, JS, Stats, and Deep Learning.

---

## 🚀 Tech Stack

- Next.js + TypeScript + TailwindCSS
- NestJS + TypeScript
- Prisma ORM
- PostgreSQL

---

## 📊 Core Features

- Daily Study Logger
- Skill-wise analytics dashboard
- Weakness detection engine
- Planned vs actual tracking
- Weekly intelligence reports

---

## 🔥 Vision

> “What gets measured gets improved.”

FocusForge builds your **feedback loop for mastery**.

---

## 📦 Setup

```bash
git clone https://github.com/your-username/focusforge.git
cd focusforge
pnpm install
```
